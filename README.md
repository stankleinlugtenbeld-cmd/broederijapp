# BroederijApp

Farm equipment floor plan management.

## Setup

1. Install dependencies: `npm install`
2. Create a Firebase project at https://console.firebase.google.com
3. Enable **Google Authentication** in Firebase Auth
4. Create a **Firestore** database
5. Copy your Firebase config into `src/firebase.ts`
6. Run: `npm run dev`

## Deploy

Push to GitHub and connect the repo to Vercel. Add Firebase config as environment variables.
